# UE5 Gameplay Automation Tests

Gameplay automation tests for Unreal Engine 5 created with Blueprints.

This repository demonstrates automated gameplay validation systems created for complex gameplay mechanics using Unreal Engine 5 Blueprints and Gameplay Ability System (GAS).

# About the Automation Tests

Each automation test validates gameplay mechanics and gameplay ability behavior directly inside Unreal Engine 5 using Blueprints.

The tests focus on:

- Gameplay Ability System (GAS)
- Gameplay Tags
- Attribute monitoring
- Runtime gameplay validation
- Ability state tracking
- Damage and health calculations
- Ability duration validation
- Debuff and buff verification
- Input simulation
- Event-driven gameplay testing
